--- layout: post title: NYPL Map Rectifier categories: - featured projects - projects tags: [] status: publish type: post published: true meta: _edit_last: '5' --- The NYPL Map Rectifier is a crowdsourcing tool for digitally aligning (or "rectifying") historical maps from the NYPL's collections to match today's precise maps. Visitors can browse already rectified maps or assist the NYPL by aligning a map. Visit the NYPL Map Rectifier (beta).
